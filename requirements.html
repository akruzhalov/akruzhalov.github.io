Требования

  • Все требования из блока "Порядок выполнения" должны быть выполнены.
  • Результат должен соответствовать макету (количество блоков, структура их наполнения).
  • Код должен быть валиден (не должно быть ошибок в валидаторе https://validator.w3.org).
  • Не должно быть ошибок оформления HTML-кода в HTMLHint.
  • Во всех лабораторных с JavaScript необходимо проверить код с помощью ESLint и исправить все возникающие ошибки.

Контент сайта

Текст (название компании, заголовки разделов, описание и т.д.)

  • Вы можете использовать текст из макета или придумать свой. Но текст обязательно должен относиться к теме сайта и быть осмысленным (нельзя использовать Lorem ipsum).

Изображения

  • Вы можете использовать изображения из макета. Ссылка на изображения указана в соответствующей лабораторной работе.
  • Или можете вставить на страницу изображения по собственному выбору.
  • Используйте бесплатные фото, например, с сайта unsplash.com.
  • Также, можно использовать Mage AI, чтобы сгенерировать собственные изображения (режим «Fast text to image» доступен бесплатно).

Размещение сайта

  • Сайт должен быть размещен на хостинге. Можно воспользоваться GitHub Pages, Netlify или хостингом Политеха.
  • Код сайта должен быть добавлен в репозиторий GitHub.

Итоговый ответ

  • В качестве выполненного задания должны быть предоставлены две ссылки:
    1. Ссылка на сайт, размещенный на хостинге;
    2. Ссылка на репозиторий с кодом на GitHub.
